- created_at
- id
- id_str
- text
- source
- truncated
- in_reply_to_status_id
- in_reply_to_status_id_str
- in_reply_to_user_id
- in_reply_to_user_id_str
- in_reply_to_screen_name
- user
- geo
- coordinates
- place
- contributors
- retweeted_status
- quoted_status_id
- quoted_status_id_str
- quoted_status
- quoted_status_permalink
- is_quote_status
- quote_count
- reply_count
- retweet_count
- favorite_count
- entities
- favorited
- retweeted
- filter_level
- lang
- timestamp_ms
